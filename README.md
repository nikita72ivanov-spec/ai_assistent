## Описание проекта

Проект представляет собой интеллектуального ассистента на основе искусственного интеллекта, разработанного специально для барбершопов. Цель решения — автоматизировать первичное взаимодействие с клиентами: отвечать на частые вопросы, консультировать по услугам и записывать на процедуры, минимизируя нагрузку на администраторов и повышая клиентский опыт.

Ассистент интегрирован в популярные каналы коммуникации (например, WhatsApp, Telegram, сайт) и работает круглосуточно, обеспечивая мгновенный отклик на запросы клиентов.

---

## Ключевой функционал

- **Осмысленное ведение диалога с помощью нейросети**  
  Ассистент понимает контекст разговора, поддерживает естественный диалог и адаптируется под стиль общения клиента.

- **Ответы на вопросы по базе знаний**  
  Все ответы формируются на основе актуальной внутренней базы знаний барбершопа: прайс-лист, описание услуг, информация о мастерах, правила отмены записи и т.д.

- **Запись клиента на встречу**  
  В ходе диалога ассистент может предложить свободные слоты, уточнить детали записи и автоматически добавить встречу в **Google Календарь** барбершопа.

---

## Используемые технологии

- **Qwen**  
  Использовалась для разработки системного промпта и структурирования базы знаний. Модель Qwen помогла сформулировать логику поведения ассистента и обеспечить точность генерируемых ответов.

- **Савви ([https://suvvy.ai/](https://suvvy.ai/))**  
  Платформа для создания мощных AI-ботов, использованная для развертывания и управления ИИ-ассистентом.  
  Преимущества использования Савви:
  - Работа на базе надёжных LLM (Large Language Models)
  - Гарантированный ответ на **100% обращений**
  - Точность ответов — **от 95%**
  - Готовая интеграция с CRM, мессенджерами и календарями (включая Google Календарь)

## Преимущества для барбершопа

- Снижение нагрузки на администраторов
- Увеличение количества записей за счёт круглосуточной доступности
- Повышение клиентской удовлетворённости за счёт быстрых и точных ответов
- Автоматизация рутинных операций (запись, напоминания, отмены)
